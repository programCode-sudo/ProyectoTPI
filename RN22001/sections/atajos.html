Atajos de Teclado en Visual Studio Code

Los atajos de teclado en VS Code pueden mejorar significativamente tu productividad. Aquí te presentamos algunos de los atajos más útiles y cómo puedes personalizarlos.

Atajos Básicos

  • Ctrl+P: Abre la barra de búsqueda de archivos, permitiendo navegar rápidamente por el proyecto.
  • Ctrl+Shift+P: Abre la paleta de comandos, donde puedes ejecutar comandos y acceder a diversas funcionalidades.
  • Ctrl+Shift+O: Muestra la lista de símbolos en el archivo actual, facilitando la navegación por el código.
  • Ctrl+`: Abre y cierra el terminal integrado.
  • Alt+Shift+F: Formatea el archivo actual según las reglas de formato configuradas.

Atajos de Edición

  • Ctrl+Space: Muestra el menú de autocompletado para el lenguaje en el que estás trabajando.
  • Ctrl+Shift+K: Elimina la línea actual sin necesidad de seleccionarla.
  • Alt+Up/Down: Mueve la línea actual hacia arriba o abajo, facilitando la reordenación del código.
  • Ctrl+/: Comenta o descomenta la línea seleccionada.
  • Ctrl+Shift+[ y Ctrl+Shift+]: Colapsa o expande bloques de código.

Atajos de Navegación

  • Ctrl+Tab: Cambia entre los archivos abiertos.
  • Ctrl+Shift+E: Abre el panel de exploración de archivos.
  • Ctrl+Shift+M: Muestra la lista de problemas en el archivo actual.
  • Ctrl+Shift+F: Busca texto en todos los archivos del proyecto.
  • Ctrl+Shift+H: Reemplaza texto en todos los archivos del proyecto.

Personalización de Atajos

Para personalizar los atajos de teclado:

  • Ve a Archivo > Preferencias > Métodos abreviados de teclado.
  • Busca el atajo que deseas cambiar y haz clic en el icono de lápiz para asignar una nueva combinación de teclas.
