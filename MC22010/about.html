¿Qué es CSS?

Un concepto clave es que una página web es realmente un documento de texto. En dicho documento se escribe código HTML, con el que se que crea el contenido de una web.
Por otro lado, existe el código CSS, que unido al código HTML permite darle forma, color, posición (y otras características visuales) a un documento web.
