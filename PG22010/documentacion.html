Documentación de Bootstrap

¿Qué es Bootstrap?

Bootstrap es un framework de código abierto para desarrollar con HTML, CSS y JS. Su diseño responsivo se adapta a diferentes tamaños de pantalla.

Instalación

Para instalar Bootstrap, puedes usar un CDN o descargar los archivos directamente desde el sitio oficial.

Componentes Básicos

Bootstrap ofrece una variedad de componentes como botones, formularios, y tablas que se pueden personalizar fácilmente.
